### 程序描述
- 1. 使用了单点交叉，每一次迭代找出两个父辈交叉生成两个子辈。
- 2. 在淘汰个体时，保护最优个体不被淘汰以保证程序的单调性，否则很容易陷入局部最优。
- 3. 共设置了20个节点，需要迭代2500轮左右趋于稳定
- 4. 经过本人测试，通过本程序求得的最短距离大约是2800；如果用纯随机搜索4万轮求得的距离大约在4000-4500之间。
